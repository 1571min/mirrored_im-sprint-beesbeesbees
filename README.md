## Sprint 진행 방법

1. 현재 repo를 자신의 Github으로 fork 합니다.
2. fork된 repo를 git clone 명령어로 자신의 local 환경으로 다운로드합니다.
3. Leanco에서 Pair Programming 전에 혼자서 진행할 부분을 확인합니다.
4. Pair분과 함께 Programming을 하며 재밌게 coding합니다.
